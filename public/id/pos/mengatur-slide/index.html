Mengatur Slide Beranda

Diterbitkan pada 26 Januari 2025, diubah pada 14 Februari 2026.

Menyiapkan konten slide

Tema Hugo Brewm memiliki bagian slide beranda yang menampilkan konten penting melalui carousel interaktif. Anda dapat menambahkan parameter cover untuk menambahkan sampul slide; parameter ini juga mengubah tata letak slide. Untuk membuat beberapa slide, Anda perlu membuat file terpisah yang berisi struktur front matter seperti ini:

---
type: slide
title: "Judul Slide"
params:
    headless: true # cegah pembuatan halaman dan alihkan ke 404.html
    target: "https://example.com" # atau alihan ke target
---

Shortcode Khusus untuk Konten Slide

Feed RSS eksternal

Bagian slide dapat mengintegrasikan konten dari sumber eksternal menggunakan shortcode RSS.

---
type: slide
title: "Postingan saya di example.com"
---

{{< rss "https://example.com/feed.xml" >}}
